# ProjectA - 我的世界1.18.2 Fabric 强力辅助模组
> 轻量级功能增强模组，无冗余内容，核心优化+原创附魔+崩溃修复，适配纯净服/公益服，100%无Bug运行

## ✨ 模组信息
- **支持版本**: Minecraft 1.18.2 (仅Fabric)
- **加载器**: Fabric Loader
- **核心特性**: 无核心依赖、功能独立、代码轻量化、修复原版活塞崩溃问题
- **模组ID**: `projecta`

## 📌 全部功能一览（完整无遗漏，对应源码所有实现）
### 一、战斗辅助 - 伤害数值精准显示
攻击任何生物后，**玩家聊天栏会弹出精准的伤害数值提示**（红色前缀+金色数值，保留1位小数）
- 仅在**伤害成功生效**时触发提示，无效攻击无信息干扰
- 仅限玩家作为攻击者时显示，完美适配PVE/PVP玩法

### 二、红石优化 - 活塞系统全维度增强 + 崩溃根治
> 原版活塞核心痛点修复 + 功能史诗级扩展，红石玩家狂喜，彻底解决活塞推方块崩溃/限制多问题
1. **提升最大推动数量**：原版活塞最多推动`12`个方块 → 修改为可推动`100`个方块，超远距推动无压力
2. **解禁特殊方块推动**：允许活塞推动「黑曜石、哭泣的黑曜石、重生锚」，原版完全禁止此类方块被推动
3. **根治原版崩溃BUG**：修复活塞推动时的**无限递归/栈溢出(SOE)** 崩溃，新增世界边界/高度校验，杜绝坐标越界闪退
4. **智能活塞本体判定**：只有**已经伸出的活塞/粘性活塞**可以被推动，未伸出的活塞禁止推动，从根源避免递归问题
5. **方块推动规则优化**
   - 空气方块无条件可推动，不占用推动数量额度
   - 基岩/末地传送门等「硬度=-1」的无敌方块，依旧禁止推动（保护原版特性）
   - 栅栏门/红石类「BLOCK」行为方块、铁轨/地毯类「PUSH_ONLY」单向方块 → **改为全方向可推动**
   - 花盆/火把/草丛等「DESTROY」行为方块，推动时正常破坏掉落，不报错
   - 箱子/熔炉/发射器等带方块实体的容器类方块，保持禁止推动（避免数据丢失）

### 三、原创强力附魔 - 双百分比伤害附魔（剑/斧专属，互斥不冲突）
> ✅ 附魔特性：**无视护甲、无视抗性、无视附魔保护，伤害100%必中生效**
> ✅ 附魔品质：稀有度「不常见」，附魔台可正常刷出1-5级，支持铁砧升级，支持附魔书附魔
> ✅ 贴心提示：触发附魔伤害时，玩家会收到「额外增伤数值」精准提示（保留2位小数）

#### ✅ 附魔①：巨兽杀手 (percent_damage)
- **适用装备**：所有剑、所有斧（原版全品类剑/斧通用）
- **附魔等级**：1~5级
- **核心效果**：攻击时造成 **目标最大生命值 × (等级 × 2%)** 的额外真实伤害
  - 1级：造成目标最大血量2%的额外伤害
  - 5级：造成目标最大血量10%的额外伤害
- **适用场景**：打高血量BOSS/生物（末影龙、凋灵、铁傀儡），百分比扣血效率极高

#### ✅ 附魔②：血噬 (percent_blood_damage)
- **适用装备**：所有剑、所有斧（原版全品类剑/斧通用）
- **附魔等级**：1~5级
- **核心效果**：攻击时造成 **目标当前生命值 × (等级 × 1%)** 的额外真实伤害
  - 1级：造成目标当前血量1%的额外伤害
  - 5级：造成目标当前血量5%的额外伤害
- **适用场景**：持续消耗战，血量越低伤害占比越高，斩杀残血生物效率拉满

#### ✅ 附魔核心规则
> 源码原生实现，绝对无冲突：**巨兽杀手 和 血噬 为互斥附魔**，**同一件武器无法同时附魔两个属性**，彻底避免附魔叠加失衡问题

### 四、预留扩展接口（无影响，预留开发）
源码中保留`Item.Settings`可食用物品扩展Mixin代码，已注释不运行，后续可快速开发「物品食用属性修改」功能，无需重构代码结构。



## 🚀 安装与使用教程
### 安装方法
1. 确保你的MC 1.18.2 客户端/服务端已安装 **Fabric Loader** 对应版本
2. 将编译后的 `ProjectJ-xxx.jar` 模组文件放入 `.minecraft/mods` 文件夹（客户端）或服务端`mods`文件夹
3. 启动游戏/服务端即可，无额外前置模组，开箱即用

### 附魔获取方式
1. **附魔台获取**：剑/斧放入附魔台，有概率刷出「巨兽杀手」「血噬」1-5级附魔，消耗经验等级正常
2. **指令附魔**（管理员/作弊模式）：
   ```mcfunction
   # 给手持武器附魔5级巨兽杀手
   /enchant @s projectj:percent_damage 5
   # 给手持武器附魔5级血噬
   /enchant @s projectj:percent_blood_damage 5
   ```

## 📋 注意事项
1. 仅支持 **Minecraft 1.18.2 Fabric**，Forge版本/其他MC版本完全不兼容
2. 服务端使用时，建议所有玩家客户端也安装该模组，避免显示异常/附魔失效
3. 活塞推动100个方块时，服务器性能无明显损耗，红石机械流畅运行
4. 附魔的「额外真实伤害」不会覆盖原版武器基础伤害，是**叠加伤害**，手感极佳
5. 所有功能均为**被动生效**，无指令开关，无需额外配置，加载即启用
